Sert à quoi ?

On peut avoir besoin de ressources en plus pour realiser notre page web.
Css permet de nombreuse chose mais si je désire utiliser des polices de caractères, des icones etc je vais avoir besoin d'ajouter c'est ressources

Font

C'est la police de caractères utiliser pour afficher notre texte.
Elle peut être de plusieurs style différents. Nativement css en propose plusieurs.

  • Serif : Les polices à empattement ont un petit trait aux bords de chaque lettre. Elles créent un sentiment de formalité et d'élégance.
  • Sans-serif : Les polices sans empattement ont des lignes épurées (sans petits traits attachés). Elles créent un aspect moderne et minimaliste.
  • Monospace : Les polices à largeur fixe oû toutes les lettres ont la même largeur fixe. Elles créent un aspect mécanique.
  • Cursive : Les polices cursives imitent l'écriture manuscrite.
  • Fantasy : Les polices fantaisie sont des polices décoratives/joueuses.

Le choix de la ou les polices à une importance réel sur le rendue de la page et l'impression qu'elle fait à l'utilisateur.
Il est généralement préconiser de ne pas avoir plus de 3 polices différentes afin de garder une certaines cohérence et facilitée de lecture.

On va trouver de nombreuse polices payantes ou gratuite sur le web. On peut notament aller regarder du coté de fonts google

Installer une police

Il suffit de la télécharger et unzip. Puis copier le fichier .ttf ( extension largement supportée ) et le placer dans notre repertoire public ( là où se trouve notre page index.html )

Maintenant il faut indiquer à notre css qu'une nouvelle police doit être prise en compte.
Attention que chaque police ajouter augmente la taille de notre fichier css et donc le temps de chargement de notre page. 

              
/* on l'importe/la définie */
@font-face {
    font-family: GreatVibes-Regular;
    src: url("../fonts/GreatVibes-Regular.ttf");
}

/* on l'utilise */
@font-face {
    font-family: GreatVibes-Regular;
    src: url("../fonts/GreatVibes-Regular.ttf");
}

Les icons

Les icons permette de rajouter du visuel et de la symbolique dans nos pages.
Il existe de nombreuse librairies et font awesome est une des plus répandue.
Pour faire simple on peut utiliser directement un cdn qui met à disposition la library
Il suffit ensuite de l'inclure depuis notre balise head en utilisant la balise link et précisant l'url.

    
<head>
  <!-- metas -->
  <meta charset="UTF-8"> <!-- utf-8 permet l'encodage de caractère accentué sans problème -->
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <!-- links -->
  <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css" crossorigin="anonymous"/>

</head>

librairies css

Une librairie réunie des classes css directement applicable sur notre html.
On pourrait s'en passer et tout crée soit même à la main mais l'utilisation d'une librairie simplifie la vie et nous permet d'avoir une base de travail.
Si je désire ajouter une librairie spécialiser dans la gestion des animations je peux utiliser animate.css
Je vais l'ajouter comme n'importe quelle ressources css directement dans le head.


<head>
  <!-- metas -->
  <meta charset="UTF-8"> <!-- utf-8 permet l'encodage de caractère accentué sans problème -->
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <!-- links -->
  <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css">

</head>

Il reste à lire la doc pour savoir comment utiliser la librairie et l'utiliser en ajoutant la classe css animate_animated et ensuite la classe de l'effet désirez animate__bounce


<div id="hero">
    <a href="/">
      <img src="assets/img/logo.png" alt="web page icon as logo of site" class="animate__animated animate__bounce">
    </a>
    <h1 class="animate__animated animate__slideInRight">Labos Web</h1>
</div>

Framework ?

Un framework est comme une super library qui résout différent problème régulièrement rencontré.
Il peut être vue comme une boite à outils cohérentes mise en place par des spécialistes
Souvent un framework css va apporter un style de base ainsi que des composants visuels.
Un composant très courant et la Card par exemple.

Installation

Il suffit généralement de suivre la doc et d'inclure le fichier css principale dans notre page.
Certains framework inclus du javascript qu'il faudrat inclure lui aussi.
Pico css est un petit framework qui apporte un certains cadre sans être trop invasif.


<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/@picocss/pico@2/css/pico.min.css"
